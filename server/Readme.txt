Server ini menggunakan node js.
Sebelum menghidupkannya, database harus ada dulu.
jalankan query yg ada di folder database/eit.sql di phpmyadmin.
setelah struktur data dibangun, lalu jalankan apache, hidupkan port MySQL.

Cara menghidupkan server adl dg menjalankan perintah berikut.
"npm start app.js"
Tapi pastikan command prompt ada di direktori tsb.
